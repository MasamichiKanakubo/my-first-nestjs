## Quick Start
### 1.setup your local branch
```bash
git close https://github.com/MasamichiKanakubo/my-first-nestjs.git
cd my-first-nestjs
```

### 2.package install
```bash
npm install 
```

### 3.start your product
```bash
npm run start
# localhost:3000/graphql
```
